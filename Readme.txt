Cara run program: sliding_window.py --image <lokasi_gambar>.jpg
atau
Run Shortcut, klik kanan Properties, ganti --image <lokasi_gambar>.jpg dgn lokasi gambar

1. Install WinPython (download versi python 3.6)
2. Install Library. Run menggunakan Winpython Prompt
- pip install imutils
- pip install opencv_python-4.0.1-cp37-cp37m-win_amd64.whl (download filenya dari https://www.lfd.uci.edu/~gohlke/pythonlibs/)
- pip install tensorflow
- pip install keras
3. Run sliding_window.py
